# Security Policy

## Supported Versions
The most recent version of OffsetInspect is supported for security-related fixes.

## Reporting a Vulnerability
If you discover a vulnerability, please open a GitHub issue describing:

1. The nature of the vulnerability  
2. Steps to reproduce  
3. The potential impact  

Do **not** include sensitive data or operational payloads in any public report.

If the issue requires private discussion, you may request a secure, direct communication method in the issue.

## Responsible Use
OffsetInspect is intended for:

- Authorized security testing  
- Research and education  
- Development and debugging  

It must not be used in unauthorized environments.  
The author assumes no responsibility for misuse or non-compliant activity.
