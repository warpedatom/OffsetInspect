# Changelog  
All notable changes to this project will be documented here.  
This project follows [Semantic Versioning](https://semver.org/).

---

## [1.0.0] - 2025-12-05
### Added
- Initial public release of OffsetInspect  
- Hex + ASCII combined display engine  
- Exact byte highlighting logic  
- Offset-to-line-number mapping  
- Caret-based source-line visualization  
- Configurable byte window context  
- Structured color-coded output  
- README, LICENSE, SECURITY, CONTRIBUTING files  
- PowerShell module manifest  
- Automated SHA-256 checksum workflow  

### Known Limitations
- No JSON output mode (planned)  
- No raw export flag (planned)  
- Offset mapping may differ for non-UTF8 encodings
